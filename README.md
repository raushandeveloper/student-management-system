# 🎓 Student Management System - Backend

This is the backend API for the Student Management System. It is built using Node.js and Express, and provides RESTful APIs to manage student data.

---

## 🚀 Features

- Add new students ➕
- Get all students 📋
- Update student details ✏️
- Delete student ❌
- REST API structure

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- CORS
- JSON (Data Handling)
